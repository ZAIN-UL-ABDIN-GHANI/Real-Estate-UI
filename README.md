## Getting Started

Follow these steps to clone and start working with this boilerplate:

1. **Clone the Repository**

   ```bash
   
   cd your-repo-name
   ```

2. **Install the Dependencies**

   ```bash
     npm i
   ```

3. **Start the Development Server**

   ```bash
    npm run dev
   ```

4. **Build for Production**

   ```bash
    npm run build
   ```
